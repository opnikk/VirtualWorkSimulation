# SerWise

SerWise is a prototype virtual work simulation platform with the goal of improving invaluable soft skills. The prototype is implemented as a web application consisting of a frontend user interface, a backend server, and AI services provided through the OpenAI API.


### Current primary features of the project:

- Voice-based AI conversations
- Scenario-guided prompting
- AI-generated feedback after each simulation
- Progress tracking for soft skill development



## How to use:

### Running the project locally:

```
Node.js
Project dependencies installed using npm install
A personal OpenAI API key with access to the Realtime API
A MongoDB Atlas database (or another compatible MongoDB instance)
A .env configuration file containing the required environment variables, including API keys and database connection details
```
